# 🌍 Air Quality Analysis
### Tools Used: Python | Pandas | NumPy | Matplotlib



## 🔍 Problem Statement
Air pollution is a critical public health issue. This project performs exploratory data analysis on air quality data to identify pollution trends, high-risk areas, and seasonal patterns across regions.



## 📁 Dataset
- **Source:** Air Quality Dataset
- **Key Fields:** Location, Date, PM2.5, PM10, NO₂, SO₂, CO, O₃



## 🛠️ Tools Used
- **Python** — core analysis
- **Pandas & NumPy** — data cleaning and manipulation
- **Matplotlib** — data visualization



## 📊 Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Pollutant trend analysis (PM2.5, PM10, NO₂, SO₂)
- Seasonal pollution pattern identification
- High-risk period and location analysis



## 💡 Key Insights
- Identified seasonal pollution spikes during winter months
- High-risk locations showed consistently elevated PM2.5 levels
- Strong correlation found between NO₂ levels and urban density



## 🚀 Future Scope
- Build a Power BI dashboard for interactive visualization
- Add predictive modeling for air quality forecasting
